Network structure dependence: A nonparametric test for network dependence variables with Thomas W. Valente. Using rewired networks as null distrubution is a common practice in network science to investigate the structure and emergence of patterns in observed networks (Carstens 2015). Studing network structural properties such as network motifs (Milo et al. 2002) has been the main motivation for this kind of permutation tests. However, as it has been mainly centered in generic classes of networks, there is no much research on this methods using social networks jointly with observable attributes. In this paper we propose a non parametric test based on network rewiring that allows to test for the structural dependence of vertex attributes upon network structure. This test can be very useful as delivers an alternative way of testing for the presence of network effects when the appropiate statistic is defined.

The impact of increasing retirement payroll-tax: Pensions, Salaries and Employment with Andrea Repetto. Using a partial equilibrium model, we analyze what would be the possible effects of rising the pensions payroll-tax in Chile over replacement rates, salaries and employment.

amcmc: Adaptive and other Markov Chain Monte Carlo methods The amcmc R package implements out-of-the-box methods for Adaptive Markov Chain Monte Carlo, and Optimal Scaling Algorithms of Random Walk MCMC. Designed to be used with the coda R package, amcmc allows running multiple chains simultaneously (parallel computing), using Parallel Tempering MCMC, and Random Walks with Reflecting Boundaries.
Code: https://github.com/USCbiostats/amcmc
CRAN: Unpublished

netdiffuseR: Network Analysis for Diffusion of Innovations Empirical statistical analysis, visualization and simulation of network models of the diffusion of innovations. The package implements algorithms for calculating network diffusion statistics such as transmission rate, hazard rates, exposure models, network threshold levels, infectiousness (contagion), and susceptibility. The package is inspired by work published in Valente, et al., (2015) DOI:10.1016/j.socscimed.2015.10.001; Valente (1995) ISBN:9781881303213, Myers (2000) DOI:10.1086/303110, Iyengar and others (2011) DOI:10.1287/mksc.1100.0566, Burt (1987) DOI:10.1086/228667; among others.
Code: http://github.com/USCCANA/netdiffuseR
CRAN: http://cran.r-project.org/web/packages/netdiffuseR/index.html

twitterreport: Out-of-the-box tools for analysis and reporting for twitter on development, this package provides functions and methods to work with twitter data. Besides of allowing the user to easily retrieve data from twitter, it comes with functions for analyzing the data such as: creating networks, identifying gender, worclouds, etc. You can checkout a live example here.
Code: http://github.com/gvegayon/twitterreport

PARALLEL: Stata module for parallel computing, Statistical Software Components (SSC) of Boston College, 2012 Inspired in the R library “snow” and to be used in multicore CPUs, parallel implements parallel computing methods through OS’s shell scripting (using Stata in batch mode) to accelerate computations. By starting a determined number of new Stata instances (child processes), this module allows the user to implement parallel computing methods without the need of having StataMP installed. Common tasks include vectorized operations, reshaping big data, running simulations (monte carlo experiments) or bootstrapping estimations. Depending on the number of cores of the CPU, parallel can reach linear speed ups significantly reducing computing wall-clock time. Finally parallel is, to the author’s knowledge, the first user contributed Stata module to implement parallel computing.
SSC :http://ideas.repec.org/c/boc/bocode/s457527.html
Code:http://github.com/gvegayon/parallel

rgexf: An R package to build GEXF graph files, The Comprehensive R Archive (CRAN), 2012 with Jorge Fábrega and Joshua Kunst. GEXF (Graph Exchange XML Format) is a language for describing complex networks structures. These files are used to import graphs into the Graph Visualization software Gephi (“The photoshop for graphs…”). rgexf allows smoothly writing GEXF files including graph attributes and dynamics.
Code: https://bitbucket.org/gvegayon/rgexf/wiki/Home
CRAN: http://cran.r-project.org/web/packages/rgexf/index.html
GEXF: http://gexf.net/format/
GEPHI: http://gephi.org/

ABCoptim: Artificial Bee Colony (ABC) Optimization, The Comprehensive R Archive (CRAN), 2013 An implementation of Karaboga (2005) Artificial Bee Colony (ABC) algorithm. This (working) version is a work-in-progress, which is why it has been implemented using pure R code. Next version will be developed using C.
Code :https://github.com/gvegayon/ABCoptim
CRAN: :http://cran.r-project.org/web/packages/ABCoptim/index.html
ABC website: http://mf.erciyes.edu.tr/abc/

googlePublicData: An R package to build Google Public Data Metadata files, The Comprehensive R Archive (CRAN), 2013 googlePublicData provides a collection of functions to set up Google Public Data Explorer data visualization tool with your own data, building automaticaly the corresponding DSPL (XML) metadata file jointly with the CSV files. All zipped up and ready to be published at Public Data Explorer.
Code: http://github.com/gvegayon/googlePublicData/
CRAN: http://cran.r-project.org/web/packages/googlePublicData/index.html
Public Data Explorer: http://www.google.com/publicdata/directory

CNU: Stata module for calculating Unitary Needed Capitals for the Chilean pensions system. This stata module, implemented using MATA, allows computing CNU; a fundamental piece for calculating pension payments for the Chilean case. This tool, suited to be used with “parallel”, comes with the full set of mortality tables and programmed withdrawal vectors to fastly compute pensions.
Code: http://github.com/gvegayon/cnu
Paper: http://www.spensiones.cl/redirect/files/doctrab/DT00057.pdf
