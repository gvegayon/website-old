PARALLEL: Stata module for parallel computing, Statistical Software Components (SSC) of Boston College, 2012 Inspired in the R library “snow” and to be used in multicore CPUs, parallel implements parallel computing methods through OS’s shell scripting (using Stata in batch mode) to accelerate computations. By starting a determined number of new Stata instances (child processes), this module allows the user to implement parallel computing methods without the need of having StataMP installed. Common tasks include vectorized operations, reshaping big data, running simulations (monte carlo experiments) or bootstrapping estimations. Depending on the number of cores of the CPU, parallel can reach linear speed ups significantly reducing computing wall-clock time. Finally parallel is, to the author’s knowledge, the first user contributed Stata module to implement parallel computing.
SSC: http://ideas.repec.org/c/boc/bocode/s457527.html

rgexf: An R package to build GEXF graph files, The Comprehensive R Archive (CRAN), 2012 with Jorge Fábrega and Joshua Kunst. GEXF (Graph Exchange XML Format) is a language for describing complex networks structures. These files are used to import graphs into the Graph Visualization software Gephi (“The photoshop for graphs…”). rgexf allows smoothly writing GEXF files including graph attributes and dynamics.
Code: https://bitbucket.org/gvegayon/rgexf/wiki/Home
CRAN: http://cran.r-project.org/web/packages/rgexf/index.html
GEXF: http://gexf.net/format/
GEPHI: http://gephi.org/

ABCoptim: Artificial Bee Colony (ABC) Optimization, The Comprehensive R Archive (CRAN), 2013 An implementation of Karaboga (2005) Artificial Bee Colony (ABC) algorithm. This (working) version is a work-in-progress, which is why it has been implemented using pure R code. Next version will be developed using C.
Code: https://github.com/gvegayon/ABCoptim
CRAN: http://cran.r-project.org/web/packages/ABCoptim/index.html
ABC website: http://mf.erciyes.edu.tr/abc/

googlePublicData: An R package to build Google Public Data Metadata files, The Comprehensive R Archive (CRAN), 2013 googlePublicData provides a collection of functions to set up Google Public Data Explorer data visualization tool with your own data, building automaticaly the corresponding DSPL (XML) metadata file jointly with the CSV files. All zipped up and ready to be published at Public Data Explorer.
Code: http://code.google.com/p/rdspl/
CRAN: http://cran.r-project.org/web/packages/googlePublicData/index.html
Public Data Explorer: http://www.google.com/publicdata/directory

CNU: Stata module for calculating Unitary Needed Capitals for the Chilean pensions system. This stata module, implemented using MATA, allows computing CNU; a fundamental piece for calculating pension payments for the Chilean case. This tool, suited to be used with “parallel”, comes with the full set of mortality tables and programmed withdrawal vectors to fastly compute pensions.
